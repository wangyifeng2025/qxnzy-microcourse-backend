# 黔西南微课平台 (QXNZY-MicroCourse)

> 面向高职院校的在线微课学习平台

## 项目简介

微课平台是一个类似中国大学慕课网/Coursera的在线学习平台，专为高职院校设计。平台支持教师按专业分类上传教学视频，学生可以按章节学习并跟踪学习进度，同时支持视频中间问答交互和章节测试作业功能。
